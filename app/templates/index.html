What is NeverAlone?

Is a notification system that can be stablished by the user easily and personalizase in so many ways. It was intended for those who experience insecurity upon returning home as a way to report a possible accident / crime that occurred during an already planned outing to the stablished contacts.

You have to set a timer at the time you plan to be back home, and if you are not able to turn it of then the alarm will:

  • Sound as reminder 10min before finish.
  • Send the meeting details to those contacts you choosed
  • Send custom message to those contacts you choosed.

NeverAlone was started as a class project for a bootcamp during de covid-19 crisis in Barcelona, you can read more about us by clicking in the following link.
